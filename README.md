# journal
fork journal from [systemd 214](https://github.com/vitalikp/systemd)

# Install
 - download [journal](https://github.com/vitalikp/journal/archive/master.tar.gz);
 - unpack archive and go into directory;
 - create **build** directory;
 - run commands:
```
	# cd build
	# cmake ..
	# make
	# sudo make install
```
